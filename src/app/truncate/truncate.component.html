Creación del Pipe Truncate

Ejemplo de uso del pipe truncate:

Texto original: "Hola"

Texto truncado: {{ "Hola" | truncate }}

Texto original: "Angular es genial"

Texto truncado: {{ "Angular es genial" | truncate:8 }}

Texto original: "Este es un ejemplo de un texto mucho más largo para probar el pipe truncate"

Texto truncado: {{ "Este es un ejemplo de un texto mucho más largo para probar el pipe truncate" | truncate:25 }}
